Git 学习笔记
============

常用 Git 命令
-------------
- git init          # 初始化一个新的 Git 仓库
- git clone <url>   # 克隆远程仓库到本地
- git status        # 查看当前工作区状态
- git add <file>    # 将文件添加到暂存区
- git commit -m ""  # 提交暂存区的更改
- git log           # 查看提交历史
- git diff          # 查看未暂存的更改
- git branch        # 列出所有分支
- git checkout -b   # 创建并切换到新分支
- git merge <branch># 合并指定分支到当前分支
- git pull          # 从远程拉取最新代码
- git push          # 将本地提交推送到远程

Git 工作流程
------------
1. 修改文件
2. git add <修改的文件>
3. git commit -m "描述本次修改的内容"
4. git push
